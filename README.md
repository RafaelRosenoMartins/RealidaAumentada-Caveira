# Realida Aumentada - Caveira
Trata-se de um protótipo com modelo 3D de uma caveira e múltiplas interações para fazer uma Realidade Aumentada (RA). Foi criado como atividade do curso de Imersão a Realidade Aumenta da instituição iRede.

REQUISITOS: 

a) Uma interação de pressionar e segurar (mousedown/mouseup) para alterar uma propriedade (como a escala ou posição);
b) Uma interação de toque rápido (gesture-tap) para disparar uma animação de "pulso" ou "balanço";
c) Se o seu novo modelo tiver uma animação interna, use o animation-mixer para tocá-la (seja em loop ou disparada por um toque).
